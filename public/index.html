Enter a url at the end of the url in the address bar. For example:

https://freecodecamp-api-osalg22.c9users.io/https://www.google.com

The app will return a JSON object with the website you put as well as a shortened version:

{"original_url": "https://www.google.com", "short_url": "https://freecodecamp-api-osalg22.c9users.io/qwerty"}

You can then use the short url and the app will redirect you to the original url.

If you submit a url in the wrong format or if you try an incorrect short url, you will get an appropriate error.
